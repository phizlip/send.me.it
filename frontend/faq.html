Frequently Asked Questions

How is this secure?

Files never touch our servers. They transfer directly between peers using WebRTC. Our server only helps peers discover each other. PeerJS uses Google's public STUN server and PeerJS's own TURN servers for NAT traversal, which helps peers connect without ever seeing your files. See the PeerJS source code for the exact servers used.

What browser should I use?

  • Chrome: Recommended for the smoothest experience.
  • Firefox: May have issues with large file transfers.
  • Safari: Loads files into memory before writing, which can cause problems with large files.

What dependencies does this use?

PeerJS for signaling and Express for static files.
